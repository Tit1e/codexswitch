# Codex Switch

[English](./README.md) | [简体中文](./README.zh-CN.md)

`Codex Switch` is a macOS menu bar app for people who use multiple Codex or ChatGPT accounts and want a simple way to switch between them.

![Codex Switch Screenshot](./assets/app.png)

## Overview

Codex Switch helps you manage multiple local Codex login profiles from the menu bar.

Instead of manually replacing auth files, you can:

- import the current account
- save multiple account snapshots
- switch the active account with one click
- view recent usage information
- label workspaces with human-friendly names

## Key Features

- Lightweight macOS menu bar experience
- One-click account switching
- Local account snapshot management
- Quick import of the currently logged-in account
- Usage refresh from the ChatGPT usage endpoint
- Workspace alias support for easier identification

## Typical Workflow

1. Log in with Codex on your Mac
2. Import the current account into Codex Switch
3. Repeat for other accounts you want to keep
4. Open the menu bar app anytime to switch accounts

## Who It Is For

Codex Switch is useful if you:

- use multiple Codex accounts
- switch between personal and team workspaces
- want a faster workflow than manually editing files
- prefer a local desktop utility instead of shell scripts

## Notes

- macOS only
- Works with local Codex auth data on your machine
- Not affiliated with OpenAI

## Open Source

This project is open source and intended as a local productivity utility for Codex users.
